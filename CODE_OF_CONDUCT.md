# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Summary

- Be respectful, inclusive, and constructive.
- Disagree with ideas; never attack people.
- No harassment, discrimination, sexualized language, or personal attacks.
- Assume good faith in others; check assumptions before escalating.

## Scope

This Code of Conduct applies to all project spaces, including the GitHub repository, issue tracker, pull requests, discussions, and any AIDRAN-affiliated communication channels.

## Enforcement

Report concerns to `hello@aidran.ai`. Reports will be reviewed in good faith. Consequences for violations may include warnings, temporary or permanent loss of contribution privileges, and removal of content.

For the full text of the Contributor Covenant 2.1, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
